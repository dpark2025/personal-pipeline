Installation & Setup

Get your registry up and running in minutes with our comprehensive installation guide

System Requirements

  • Operating System: Linux, Windows, or macOS
  • Docker: Version 20.10 or later
  • Memory: 4GB RAM minimum, 8GB recommended
  • Storage: 10GB available disk space
  • Network: Outbound internet access for initial setup

Quick Installation

bash 
# Download and install the registry
curl -fsSL https://registry.company.com/install.sh | bash

# Start the registry services
docker-compose up -d

# Verify installation
./bin/registry-cli health-check

Configuration Options

Parameter Default Description
REGISTRY_PORT 5000 Registry service port
WEB_UI_PORT 8080 Web interface port
STORAGE_BACKEND filesystem Storage driver (filesystem, s3, azure)
